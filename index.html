Sobre la comunidad

¿Qué es Python?

Python es un lenguaje de programación interpretado, de tipado dinámico y multiplataforma, con una sintaxis que favorece el código legible. Multiparadigma, fácil de aprender y con una comunidad global enorme, se usa desde scripting hasta ciencia de datos, web y automatización.
